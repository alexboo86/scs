# ⚠️ ВАЖНО: Безопасность

## Пароль в скриптах

**ВНИМАНИЕ:** Пароль root встроен в скрипты для удобства, но это **НЕ БЕЗОПАСНО** для продакшн использования!

### Риски:
- ❌ Пароль хранится в открытом виде в файлах
- ❌ Любой, кто получит доступ к файлам, увидит пароль
- ❌ Пароль попадет в Git историю (если закоммитите)

### Рекомендации:

#### Вариант 1: SSH ключи (ЛУЧШИЙ) ⭐

```powershell
# Настройте SSH ключи один раз
.\setup-ssh-keys.ps1

# После этого пароль не нужен!
.\deploy.ps1
```

#### Вариант 2: Переменная окружения

Удалите пароль из скриптов и используйте переменную:

```powershell
$env:VPS_PASSWORD = "vE1x~!56#8nF36p8dLGW"
.\deploy.ps1
```

#### Вариант 3: Файл с паролем (не коммитить в Git!)

Создайте файл `.vps_password` (добавьте в .gitignore):

```powershell
# В скрипте:
$VPS_PASSWORD = Get-Content .vps_password -Raw
```

### Что сделать СЕЙЧАС:

1. **Настройте SSH ключи** (рекомендуется):
   ```powershell
   .\setup-ssh-keys.ps1
   ```

2. **Или удалите пароль из скриптов** и используйте переменную окружения

3. **Добавьте скрипты с паролем в .gitignore** (если еще не сделано)

4. **НЕ коммитьте скрипты с паролем в публичный репозиторий!**

## Текущее состояние

Пароль встроен в скрипты для удобства разработки, но:
- ⚠️ Используйте только на вашем локальном компьютере
- ⚠️ НЕ делитесь этими файлами
- ⚠️ Настройте SSH ключи для продакшн использования
