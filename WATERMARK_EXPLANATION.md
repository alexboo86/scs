# Объяснение водяных знаков

## Статический водяной знак

**Что это:** Изображение (логотип, эмблема), которое накладывается на все страницы одинаково.

**Характеристики:**
- Не меняется в зависимости от пользователя
- Одинаковый для всех страниц документа
- Обычно это PNG/JPEG файл с прозрачностью
- Пример: логотип школы, эмблема компании

**Настройки:**
- `static_watermark_enabled` - включить/выключить
- `static_watermark_id` - ID статического водяного знака из БД (таблица `static_watermarks`)
- `static_watermark_scale` - размер относительно страницы (0.1 - 1.0)
- `opacity` - прозрачность (0.0 - 1.0)
- `position` - позиция на странице (center, top-left, top-right, bottom-left, bottom-right)

**Как использовать:**
1. Загрузите изображение логотипа через админ-панель (функционал загрузки нужно добавить)
2. Выберите этот водяной знак в настройках (`static_watermark_id`)
3. Настройте размер и позицию

## Динамический водяной знак

**Что это:** Текст, который генерируется индивидуально для каждого пользователя и сессии просмотра.

**Характеристики:**
- Меняется в зависимости от пользователя
- Может содержать персональную информацию (email, IP)
- Может содержать информацию о сессии (время, номер страницы)
- Генерируется на лету при каждом запросе страницы

**Настройки:**
- `dynamic_watermark_enabled` - включить/выключить
- `show_user_email` - показывать email пользователя
- `show_user_id` - показывать ID пользователя
- `show_ip_address` - показывать IP адрес
- `show_timestamp` - показывать время просмотра
- `show_page_number` - показывать номер страницы
- `custom_text` - дополнительный текст
- `opacity` - прозрачность (0.0 - 1.0)
- `font_size` - размер шрифта (12 - 200)
- `color_r`, `color_g`, `color_b` - цвет текста (RGB, 0-255)
- `position` - позиция на странице

**Пример текста:**
```
"Моя школа | User: student@example.com | IP: 192.168.1.100 | Page: 1 | 2026-01-03 17:30"
```

## Разница

| Характеристика | Статический | Динамический |
|---------------|-------------|-------------|
| Тип | Изображение | Текст |
| Изменяется | Нет | Да (для каждого пользователя) |
| Содержит | Логотип/эмблема | Персональная информация |
| Применение | Одинаково для всех | Индивидуально |
| Защита | Брендинг | Отслеживание утечек |

## Рекомендации

**Используйте оба типа вместе:**
- Статический - для брендинга (логотип школы)
- Динамический - для защиты от утечек (email пользователя, IP)

**Пример комбинации:**
- Логотип школы в центре (статический, прозрачность 0.2)
- Email пользователя внизу справа (динамический, прозрачность 0.3)
